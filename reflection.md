# Reflection – Dart Fundamentals Assignment

**What was easiest for you?**  
Understanding loops and conditionals in Task 1 felt very natural coming from other languages. Writing the analysis logic was straightforward.

**What was most challenging?**  
Getting comfortable with async/await syntax and deciding where to put `await`. Also creating a custom exception class was new.

**One thing you learned about Dart:**  
I liked how clean the `Future` + `async/await` pattern is compared to JavaScript promises with `.then()`. Named constructors and custom exceptions are also very elegant.

**Did you attempt any bonus features?**  
Yes — added mostly-positive/mostly-negative detection and range warning in Task 1, and a `powerWithDelay()` method + chained async operations in Task 2.

**How much time did it take?**  
Around 3.5–4 hours total (including testing, refactoring and writing reflection).

**Any feedback about the assignment?**  
Very good progression from basic loops → OOP + exceptions → async. Would love to see file I/O or List methods (map, where, fold) in a future assignment.
